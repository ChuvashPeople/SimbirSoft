# SimbirSoft
В данном приложении не используются какие-либо сторонние библиотеки.
После запуска приложение нужно будет ввести ссылку на сайт. Программа выводит уникальные слова и их количество в консоль.
# Принцип работы программы:
1.	При помощи экземпляра класса WebClient скачивается html-страница. Текстовый файл сохраняется по адресу   “ …\SimbirSoft\SimbirSoft\SimbirSoft\bin\Debug\net5.0” под именем newHtml.
2.	Затем, каждая строка записывается в отдельный текстовый файл. Первая строка расположена в файле под именем “1” и т.д..
3.	После, в методе Merge, строки соединяются для того, чтобы далее программа смогла оставить только текст. 
4.	В строке, при помощи регулярных выражений убираются цифры, спецсимволы html и javascript-код.
5.	Далее, использую цикл, программа берет текст, находящийся между тегами. При помощи списка разделителей удаляются знаки препинания и прочие символы. Полученные слова заносятся в список. 
6.	Пункты 3,4,5 повторяются до тех пор, пока не будут обработаны все строки.
7.	При помощи LINQ программа группирует слова.
